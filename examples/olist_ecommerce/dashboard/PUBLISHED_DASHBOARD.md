# Published AI/BI Dashboard

The Olist E-Commerce AI/BI dashboard is published and shareable:

[Open the live dashboard](https://dbc-5a674036-8eaa.cloud.databricks.com/dashboardsv3/01f173eb9b821ef9b5cf8e6c8ec78028/published?o=7474648785966975)

## Demo Scope

- 29 interactive widgets
- 8 dashboard pages
- Run-as-owner sharing for external demo viewers
- Built from the Olist gold and diagnostic Delta tables
- Designed for marketplace operations, delivery performance, customer experience, and commercial mix analysis

## Portfolio Story

This dashboard is the visual layer of the Genie deployment kit:

1. Public Olist CSV files are loaded into Databricks.
2. Bronze tables are created from raw source files.
3. Gold and diagnostic tables add business-ready metrics.
4. Genie semantic metadata, SQL examples, snippets, and benchmarks are stored in Git.
5. The published AI/BI dashboard turns the same governed tables into an interactive executive demo.
6. The generated action playbook extends the demo into recommendation and action-planning workflows.

## Suggested Review Flow

Use the dashboard to inspect:

- order value and order volume trends
- late delivery rate patterns
- review score quality
- product category Pareto concentration
- customer-state concentration
- delivery impact on customer experience
- target gaps for reaching a 4.2 average review score

Then use Genie questions and the generated playbook to move from diagnosis to action planning.
