# 02 — Reference Asset Map
> **Last updated:** 2026-06-19
> Map of all reference/library assets (not directly deployed to room).

## instruction_library/ — Canonical Corpus

### corpus/
- **example_sql/** — Full superset of all possible example SQL
  - Files: `NN_slug.yml` format
  - Count: {{LIBRARY_SQL_COUNT}} (vs {{EXAMPLE_SQL_COUNT}} active)

- **filters/** — Full superset of all filter snippets
  - Count: {{LIBRARY_FILTER_COUNT}} total

- **measures/** — Full superset of all measure snippets
  - Count: {{LIBRARY_MEASURE_COUNT}} total

- **dimensions/** — Dimension helper snippets (repo-only)
  - These never get deployed to the room

### activation/
- `filters.active.yml` — Allowlist of active filter IDs
- `measures.active.yml` — Allowlist of active measure IDs
- `dimensions.active.yml` — Allowlist of active dimension IDs
- `example_sql.active.yml` — Allowlist of active example SQL IDs
- `limits.yml` — Capacity budget configuration

### reduction/
- Merge/isolation decision records for deduplication analysis

### isolated/
- `deactivated.yml` — IDs of deliberately deactivated snippets
- Preserves ID history for audit trail
