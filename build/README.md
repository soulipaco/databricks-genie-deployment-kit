# Build — Generated Outputs

This directory contains generated outputs from automation scripts.

## Files

| Pattern | Source | Purpose |
|---------|--------|---------|
| `eval_batch.yml` | `scripts/prepare_eval.py` | Benchmark evaluation batch |
| `eval_*.yml` | Manual evaluation runs | Evaluation results |
| `eval_summary.yml` | `scripts/analyze_benchmarks.py` | Analysis summary |
| `health_check_*.txt` | `skills/health_check` | Health check reports |
| `deployment_readiness/` | `scripts/prepare_eval.py` | Readiness reports |

## Rules

- **NEVER edit files in this directory** — they are regenerated by scripts
- They can be safely deleted and regenerated
